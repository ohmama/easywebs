# 网站设计提示词指南

## 设计风格关键词（视觉）

### 布局结构
```
sticky navigation bar
two-column hero layout
card-based sections
clear visual hierarchy
generous white space
consistent grid
```

### 视觉风格
```
clean
minimal
professional
light background
subtle shadows
soft border radius
warm off-white surface colors
```

### 字体
```
sans-serif body font
serif accent for headings
high contrast type scale
tight letter-spacing on headings
```

### 颜色
```
neutral warm palette
single accent color
muted secondary text
dark footer
```

### 交互
```
subtle hover states
gentle scroll reveal
no parallax
button fill on hover
focus ring on form inputs
```

### 组件
```
pricing card with dark header
feature checklist with icon
trust badge row
numbered process timeline
inline form with label above input
```

## 一句话提示词模板

> "Clean, professional SaaS-style layout. Two-column hero with a pricing card on the right. Warm off-white background, single blue accent color, subtle card shadows. Sans-serif body with serif headings. Sticky nav, trust bar below hero, card-based service grid, numbered vertical timeline for process, dark banner for pricing highlight. Subtle scroll reveal only. No gradients, no decorative illustrations, no experimental layouts."

## 转化向内容提示词模板

```
Design a clean, conversion-focused website for [服务类型] targeting [目标客群].

PRIMARY GOAL: [一句话说清核心目标]

Users should understand within 5 seconds:
- What you do
- How much it costs (roughly)
- What to do next

Structure: hero / trust bar / services / process / pricing / about / contact

MESSAGING:
- Headline: [具体标题]
- Subheadline: [具体副标题]
- CTAs (行动按钮): [具体按钮文字], [备选按钮]
- Use words: simple, affordable, fast, clear
- Avoid words: creative, unique, premium, innovative

PRICING:
- Show starting price prominently in hero
- Show price range [X–Y]
- Repeat "no hidden costs" near every CTA
- Address 3 user fears: too expensive / too complicated / getting ripped off

DESIGN:
- 70% clear structure, 30% visual refinement
- Subtle animations only (scroll reveal, hover states)
- No experimental navigation
- Mobile-first

AVOID:
- Abstract or artistic layouts
- Confusing navigation
- "Learn more" or "Explore" CTAs
- Any copy that sounds like a design agency
```

---

## 注意事项

**提示词里出现这些词，AI 会优先服务"美学"而不是"转化"：**
- editorial
- deconstructivist
- experimental
- cutting-edge
- premium design

**如果两者都想要，加这一句：**
> "Elevate the design with refined typography and spacing, but never at the expense of clarity or conversion."

---

## 名词解释

| 术语 | 含义 |
|------|------|
| CTA | Call to Action，行动按钮，例如"Get a quote"、"See pricing" |
| Hero | 网页首屏区域 |
| Trust bar | 信任信息条，通常在 Hero 下方，列出简短的信任背书 |
| Scroll reveal | 滚动触发的淡入动画 |
| Two-column layout | 两栏布局 |
